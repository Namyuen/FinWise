# FinWise

## Overview
FinWise is a static educational website designed to improve financial literacy by providing learning resources, interactive quizzes, and practical financial tools.

## Motivation
This project was created to make financial education more accessible through a simple, user-friendly web application.

## Features
- Educational articles on personal finance
- Interactive quizzes
- Financial calculators and tools
- Responsive design for desktop and mobile

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
index.html – Homepage
about.html – About page
learn.html – Learning resources
quiz.html – Interactive quizzes
tools.html – Financial tools
resources.html – Additional resources
style.css – Styling
script.js – Interactive functionality

## Future Improvements
- User accounts and progress tracking
- Backend database integration
- Personalized learning paths
- Accessibility improvements
